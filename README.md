# Architecture Computational Tools Library

This repository was created as part of the research activities of the Department of Architecture, University of Thessaly. Its purpose is to collect, share, and expand computational tools that support the design and research workflow of architecture students.

## Purpose

The tools gathered here aim to improve the efficiency of architectural workflows — from parametric modeling and environmental analysis to design optimization. The repository is addressed primarily to architecture students who want to integrate computational methods into their academic work, whether at an early stage or at an advanced level.

Beyond individual use, this library is meant to grow collectively. Every student who has developed a tool, a script, or a workflow that solved a real problem in their work is encouraged to contribute. The goal is not to build a perfect archive, but a living resource that reflects the actual needs and discoveries of the student community.

## Contents

The repository currently includes tools related to:

- Parametric geometry generation in Grasshopper
- Environmental analysis workflows using Climate Studio
- Window optimization and adaptive facade logic
- Python scripts for Grasshopper automation

New categories will be added as the library grows.

## How to Use

1. Browse the folders and find the tool relevant to your project
2. Download the `.gh` or `.pdf` file
3. Open it in Rhino/Grasshopper
4. Each file includes internal notes explaining inputs, outputs, and how to connect it to your model

## How to Contribute

If you want to add a tool or improve an existing one:

1. **Fork** this repository
2. Add your file with a short description inside the folder
3. Submit a **Pull Request** with a brief note on what it does and what problem it solves
4. The maintainer will review and merge it

All contributions are credited to their authors.
